Ez a program a személyes költségvetések menedzselésére való.

Az bejegyzett költségek (azok azonosítója, neve, értéke Forintban, kategória száma, bejegyzés dátuma és hozzászólása) 
a PQ7I00/DataFiles/Spendings mappában találhatók.

A bejegyzett adatok nem kerülnek fel a távoli repository-ba, 
ezért teszteléshez érdemes a PQ7I00/DataFiles/InitSpendingTestData mappából átmásolni 
a json fájlokat a PQ7I00/DataFiles/Spendings mappába, mivel a program ebben keresi az adatokat a futása folyamán.
